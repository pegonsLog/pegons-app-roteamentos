Como usar:
  1. Clique no campo acima para selecionar um arquivo KML do seu computador
  2. O sistema irá processar o arquivo e extrair os dados automaticamente
  3. Os dados extraídos serão exibidos em uma tabela
  4. Clique em "Exportar para Excel" para baixar os dados em formato .xlsx
Dicas:
  • O arquivo KML deve conter tags <Placemark> com informações dos locais
  • O nome é extraído da tag <name>
  • O endereço pode estar na tag <address> ou na descrição
  • O turno é identificado automaticamente na descrição (palavras-chave: manhã, tarde, noite, etc.)
  • Se alguma informação não for encontrada, valores padrão serão usados
